Tactics

Abstract

Christine.Tactics implements a minimal Coq-style tactical proof engine in Elixir (lib/christine/tactics.ex). Every tactic is a pure function on the ProofState struct. It either returns {:ok, new_state} or {:error, reason, state}. The engine builds the final CIC proof term via explicit reconstructor closures — no evars, no unification variables, no backtracking.

Proof State

defmodule Christine.Tactics.ProofState do
  defstruct [
    :name,          # theorem name (String)
    :target,        # original goal type
    :env,           # typechecker environment
    :goals,         # list({context, goal_type})
    :proof_term,    # completed term when goals == []
    :reconstructor  # fn([subproofs]) -> term
  ]
end

The reconstructor is updated by every tactic to wrap the sub-proof(s) into the correct constructor (Lam, App, etc.).

intro

intro x works only when the current goal normalizes to a Pi type ∀ x : A, B. It adds the binding x : A to the context, replaces the goal with B, and updates the reconstructor to wrap the sub-proof in Lam x A body.

Error: :not_a_pi if the goal is not a dependent function type.

Theorem plus_O_n : ∀ n : Nat, plus O n = n.
Proof.
  intro n.
  reflexivity.
Qed.

intros

intros x y z repeats intro for each name. Implemented via the helper do_intros. It chains the reconstructor updates so the final term has nested Lam binders.

Theorem app_nil : ∀ A (l : List A), app l nil = l.
Proof.
  intros A l.
  induction l.   -- (once induction is added)
  - simpl. reflexivity.
  - simpl. rewrite IHl_0. reflexivity.
Qed.

exact

exact t parses the expression t in the current context, infers its type, and checks that it equals the goal (or is Any). If successful, it closes the goal and updates the reconstructor to insert the term directly.

Error: :type_mismatch with the inferred and required types.

Theorem plus_O_n : ∀ n : Nat, plus O n = n.
Proof.
  intro n.
  exact (Refl Nat n).
Qed.

simpl

simpl replaces the current goal with its βι-normal form using Typechecker.normalize. It does not change the proof term — only the goal presentation.

Theorem plus_O_n : ∀ n : Nat, plus O n = n.
Proof.
  intro n.
  simpl.         -- reduces plus O n to n
  reflexivity.
Qed.

reflexivity

reflexivity normalizes the goal. If it is an equality a = a, it closes the goal with Refl and updates the reconstructor accordingly. The equality check uses syntactic Typechecker.equal? after normalization.

Theorem plus_O_n : ∀ n : Nat, plus O n = n.
Proof.
  intro n.
  reflexivity.
Qed.

ring

ring normalizes the goal to an equality, converts both sides to polynomials via to_poly + poly_normalize, and solves if the polynomials are identical. It closes the goal with a placeholder term ring_solved.

Error: :not_an_equation or :ring_mismatch.

Theorem ring_example : (x + 2*y) + (3*x + y) = 4*x + 3*y.
Proof.
  ring.
Qed.

apply

apply H or apply H with x:=t looks up hypothesis H, matches its type against the goal using match_goal, generates subgoals for missing arguments, and builds an App node in the reconstructor. Named with bindings are evaluated and substituted.

Error: :variable_not_found, :cannot_apply.

Theorem plus_comm : ∀ n m : Nat, plus n m = plus m n.
Proof.
  intros n m.
  apply plus_comm_helper.   -- assuming helper is in context
Qed.

Implementation

All tactics are defined in lib/christine/tactics.ex. The dispatcher is the apply_tactic function with pattern matching on the parsed tactic tuple. New tactics are added by extending parse_tactic and the main case clause.
